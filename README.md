# Blog
## 基于React+NodeJs+Express+MongoDB+antd框架实现简易博客
### 目前实现的功能有:用户登录注册 用户信息管理 内容管理 多级评论 文章/评论分页 头像上传等 
#### **本地运行**
##### 安装项目
>git clone https://github.com/RayAidas/Blog.git

>cd Blog

>npm install webpack -g

>npm install webpack-dev-server -g

>npm install 
##### 服务端
>node server/bin/www
##### 前端
>npm run build

>npm run dev
