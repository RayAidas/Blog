# Blog
## 基于React+NodeJs+Express+MongoDB+antd框架实现简易博客
### 目前实现的功能有:用户登录注册 修改用户信息 内容管理 用户评论等 
#### **本地运行**
##### 安装项目
>git clone https://github.com/RayAidas/Blog.git

>cd Blog
##### 服务端
>node server/bin/www
##### 前端
>npm install webpack -g

>npm install webpack-dev-server -g

>npm install 

>npm run build

>npm run dev
